# Imágenes de Homepage

Esta carpeta contiene las imágenes principales de la página de inicio.

## Archivos con nombres consistentes:

- **hero.jpg** - Imagen principal del hero section (1200x800px)
- **featured1.jpg** - Primera imagen destacada del portafolio (800x600px)
- **featured2.jpg** - Segunda imagen destacada del portafolio (800x600px)
- **featured3.jpg** - Tercera imagen destacada del portafolio (800x600px)
- **style1.jpg** - Ejemplo de estilo Realismo (600x600px)
- **style2.jpg** - Ejemplo de estilo Geométrico (600x600px)
- **style3.jpg** - Ejemplo de estilo Japonés (600x600px)
- **style4.jpg** - Ejemplo de estilo Blackwork (600x600px)

## Importante:
- Los nombres de archivos se mantienen consistentes para evitar delays de carga
- Al actualizar imágenes desde el admin, se reemplaza el archivo existente
- Formatos soportados: JPG, PNG, WebP
- Las imágenes se redimensionan automáticamente al formato correcto
